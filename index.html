Configuración inicial
Para compartir esta app con tu pareja en tiempo real, necesitas conectar Firebase (base de datos gratuita de Google). Solo toma 5 minutos.
Paso 1
Ve a console.firebase.google.com con tu cuenta Google y crea un proyecto nuevo. Nómbralo finanzas-parra.
Paso 2
Dentro del proyecto ve a Realtime Database → Crear base de datos → Modo de prueba → Copiar la URL que aparece (termina en .firebaseio.com).
Paso 3
Ve a Configuración del proyecto → Agregar app → Web → Copia el objeto firebaseConfig que aparece.
Paso 4
Abre el archivo finanzas-familia.html en un editor de texto, busca AIzaSyCJxNItddnLwp9GCCjp30_Gvo5J99E4P5U y reemplaza con tus datos. Listo.
Paso 5 — Publicar
Sube el archivo a GitHub Pages (gratis) y comparte el link con tu pareja. Ambos ven los mismos datos.
💡 Una vez configurado, solo aparece esta pantalla si el archivo no tiene las credenciales. El link publicado no la mostrará.
